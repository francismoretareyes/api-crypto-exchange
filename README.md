# 🪙 CryptoExchange API
**Autor:** Francis Moreta Reyes
## 📖 Descripción del proyecto
CryptoExchange es una API RESTful construida en **Node.js, Express y TypeScript** con base de datos en **MongoDB**. Simula el comportamiento de un Exchange de Criptomonedas donde un usuario al autenticarse puede gestionar su cartera de activos, registrar compras/ventas de manera validada, y obtener un cálculo de sus balances y agregaciones cruzando los datos en tiempo real con la API externa **CoinCap**.
La aplicación incorpora una arquitectura multicapa para evitar manipulaciones en el servidor por parte de clientes maliciosos, además de integrar un sistema de *Fallback* en caché tolerante a fallos de red por parte de las APIs externas.
